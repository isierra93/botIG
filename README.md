## Ejectuar localmente

Clonar el proyecto

```bash
  git clone https://github.com/isierra93/botIG.git
```

Ir al directorio del proyecto

```bash
  cd botIG
```

Instalar dependencias

```bash
  npm install
```

Ejecutar el servidor

```bash
  npm start
```
